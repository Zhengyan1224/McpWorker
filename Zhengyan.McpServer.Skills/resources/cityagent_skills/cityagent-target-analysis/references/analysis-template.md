您好，[对象名称][范围说明]分析已完成，具体内容如下：

一、问题类型梳理

1. [指标名称]：[数量] [单位]
   [问题归类 + 主要表现 + 与对象场景的关系]

2. [指标名称]：[数量] [单位]
   [问题归类 + 主要表现 + 与对象场景的关系]

3. [指标名称]：[数量] [单位]
   [问题归类 + 主要表现 + 与对象场景的关系]

问题叠加：[仅当数据足够支撑时再写；没有依据就删掉这一行]

二、分布特征分析

1. [空间分布或对象分布]
2. [集中原因]

4. [联动影响或扩散效应]

三、潜在风险分析

1. [核心风险]
2. [次生风险]
3. [治理推进风险]

四、改造优先级与统筹建议

1. 最高优先级
   [先做什么，期限或处置方向是什么]

2. 次高优先级
   [中短期改造重点]

3. 统筹实施
   [长效机制、统筹联动、后续运营维护]

## 使用要求

- 小区对象优先写 2 到 3 个重点指标；社区对象优先写聚合后的前 3 个指标
- 没有充分依据时，删除“问题叠加”或缩减“分布特征分析”的条目，不强行凑满
- 允许根据数据量压缩条目，但四个一级部分要保留
