# 常用指标分析映射

以下内容用于把指标名称转换成更稳定的分析语言。命中时优先复用；未命中时按相近类别保守处理。

## 住房安全类

- 关键词：结构安全隐患、围护安全隐患、楼道安全隐患、燃气安全隐患
- 问题归类：安全耐久、生命财产安全底线
- 常见表述：房屋本体安全、公共通行安全、燃气使用安全
- 风险侧重：坍塌、坠落、火灾、爆炸、通行伤害
- 优先建议：先鉴定、先排险、先临时防护，再纳入系统整治

## 住房设施与功能类

- 关键词：管线管道破损、水质不达标、适老化改造、节能改造、数字化改造
- 问题归类：功能完备、设施老化、民生服务短板
- 常见表述：给排水、电力、无障碍、居家安全、设施更新需求
- 风险侧重：渗漏、短路、滑倒、生活不便、老年人使用障碍
- 优先建议：与结构修复、公共管网改造、无障碍改造统筹实施

## 社区公共服务类

- 关键词：养老服务设施、婴幼儿照护服务设施、幼儿园、公共活动场地
- 问题归类：公共服务设施短板
- 常见表述：一老一小服务供给不足、设施缺配、功能不全
- 风险侧重：居民基础服务缺口、生活便利度下降、社区吸引力走弱
- 优先建议：先补功能缺项，再补服务运营机制

## 社区环境与治理类

- 关键词：步行道、物业管理、垃圾分类、电动自行车充电、严重易涝
- 问题归类：市政配套、社区治理、日常运维缺位
- 常见表述：通行环境不足、维护机制薄弱、公共空间管理不到位
- 风险侧重：摔倒、交通冲突、消防隐患、雨天积水、治理失序
- 优先建议：先修安全短板，再建巡检维护和责任落实机制

## 街区与城区类

- 关键词：老旧街区、中学服务半径、公园绿化活动场地、多功能运动场地、黑臭水体、易涝积水点、轨道站点周边覆盖通勤比例
- 问题归类：空间服务覆盖不足、基础设施韧性不足、存量更新需求
- 常见表述：服务盲区、公共空间品质不足、城市运行韧性短板
- 风险侧重：民生服务不均衡、区域活力下降、生态环境恶化、通勤效率低
- 优先建议：按“安全与韧性优先、公共服务补短板、存量空间活化”三类统筹推进

## 写作规则

- 归类优先依据指标关键词，不要过度延伸。
- 风险和建议必须与指标类型一一对应，不要给所有指标套同一套空泛建议。
- 没有直接依据时，使用“可能”“通常与……有关”“易导致”这类保守表述。
