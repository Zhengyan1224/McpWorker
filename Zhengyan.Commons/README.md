# Zhengyan.Commons

`Zhengyan.Commons` 是通用工具库，被多个服务和库项目引用。

## 目标框架

```text
net8.0
net9.0
```

## 主要能力

- `TextProcessPipeline` / `ITextHandler`：文本处理管道。
- `ExtractValidCharFromHtmlHandler`：HTML 文本清洗辅助。
- `LRU`、`HashLinkedList`：缓存和链表工具。
- `ClientCertificateProvider`：简单的客户端凭据校验辅助。
- `XXTea`：XXTEA 相关工具。
- `SystemUtils`、`TextUtils`、`AdvancedConsole`：常用工具方法。

这个项目不单独运行。
